# 学习资料

这个章节整理了 Rust 语言的学习资源，包含官方文档、工具文档、在线工具、版本历史以及社区教程和视频教程等，供 Rust 学习者参考使用。

如果你有更好的资源推荐，欢迎提交 [pull request](https://github.com/georgetime1970/rust8.cn/pulls) 来补充和完善这个资源列表，也欢迎提交 [issue](https://github.com/georgetime1970/rust8.cn/issues) 来反馈。
